# 🚢 Titanic Visualization Dashboard (Task 4)

This project presents a **Mini Data Visualization Dashboard** built using the Titanic dataset.  
The goal is to communicate key insights through clear and meaningful visualizations.

---

## 📌 Objectives

- Perform data cleaning and preprocessing  
- Apply feature engineering (Age Groups, Family Size)  
- Build multiple visualizations in a single notebook  
- Present insights using a storytelling approach  

---

## 🧹 Data Cleaning

- Filled missing values in Age using mean  
- Dropped Cabin column due to excessive missing data  
- Filled missing Embarked values  

---

## 🔧 Feature Engineering

- Created Age Groups (Child, Teen, Adult, etc.)  
- Calculated Family Size (SibSp + Parch)  

---

## 📊 Dashboard Visualizations

- Survival rate by Gender  
- Survival rate by Passenger Class  
- Age Distribution Histogram  
- Correlation Heatmap  
- Survival by Family Size  

---

## 💡 Key Insights

- Females had significantly higher survival rates  
- First-class passengers survived more  
- Most passengers were between 20–40 years  
- Smaller families had better survival chances  
- Passenger class strongly influenced survival
